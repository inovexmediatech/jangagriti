# Shree Jana Jagriti Secondary SchooSchool Website

This repository has been converted to a frontend-only site. The Express server and SQLite backend were removed.

Open `new.html` (or `index.html`) in a browser to view the site. All dynamic behaviors run in the browser without a server:

- Facilities and events are provided as local data
- Gallery uploads are handled client-side (in-memory) and do not persist to disk
- Contact form opens the user's email ccclient via `mailto:`

To run locally, open the HTML file in your browser or use a simple static server such as `npx serve`.
